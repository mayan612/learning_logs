# learning_logs
这个web项目是用django做出来的

## 功能
- 用户注册、登录
- 每个用户都有自己的数据
- 使用 /admin 以管理你的网站
- 更多功能请自行测试

- 可以在用户里面管理用户

## 需要的环境
- Python3.x
- django

## 安装步骤

1. 克隆此仓库到一个合适的文件夹
2. 安装Python3.x和pip
3. 安装django：pip install django
4. 启动服务器: python3 manage.py runserver <你的ip>:<端口>
5. 从你设置的地址看效果吧
- 注意: 你必须在learning_Log/settings.py里的ALLOWED_HOSTS写上你的ip
